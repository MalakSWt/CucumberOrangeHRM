# 🧪 OrangeHRM Test Automation Framework

## 📌 Overview
This project is a complete **Test Automation Framework** built for the OrangeHRM demo application.  
It covers **end-to-end employee management workflows** using modern automation best practices.

The framework is designed with scalability, maintainability, and readability in mind.

---

## 🚀 Tech Stack
- **Java**
- **Selenium WebDriver**
- **Cucumber (BDD)**
- **TestNG**
- **Maven**

---

## 🧩 Framework Design
- Page Object Model (POM)
- Behavior-Driven Development (BDD)
- Reusable utilities (WaitUtils, DriverFactory)
- Clean separation between:
  - Pages
  - Step Definitions
  - Hooks
  - Utilities
